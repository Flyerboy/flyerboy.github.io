PHPStorm 的基本配置

发表于 2017-11-08

一直使用 PhpStorm 敲代码，更新了 2017.2 版本之后，有几个不习惯的地方，也可以说是强迫症，反正自己感觉不习惯，修改了配置，在此做一下记录。

函数或方法前的 hints

这个功能其实很好的，个人原因不想让他显示出来，所以设置隐藏。找到下面的选项，将其选中取消。

单行注释在行首

感觉注释在行首的话，很破坏代码的美感。

编码区域的竖线

自我感觉这条线没什么用，有点多余，将其隐藏。找到下面的选项，将其选中取消。

修改的文件 tab 上加 *

这个是为了区分当前文件是否做了修改，找到下面的选项，打钩。

左侧菜单颜色同步

如果使用黑色背景的话，左侧菜单颜色和编码区域颜色不一致，看着很别扭，这就需要安装一个 color ide 的插件，以前在 PhpStorm 的插件列表中可以搜到，直接安装就行，更新版本之后搜了没找到，就手动下载安装。
