C 语言实现栈

发表于 2016-12-11

定义

  1. 栈 stack 是限定仅在尾部进行插入和删除操作的线性表
  2. 我们把允许插入和删除的一端称为栈顶 top，另一端称为栈底 bottom，不包含任何数据的元素为空栈
  3. 栈又称为后进先出 Last in First Out 的线性表，简称 LIFO 结构
  4. 当 top 为空时，表示空栈

注：

  1. 首先栈是一种特殊的线性表，其具有线性表的关系，前驱和后继，而且特性属性，从栈顶进，从栈顶出
  2. 栈底是固定的，最先进栈的在栈底 bottom
  3. 栈的插入操作称为 入栈 push
  4. 栈的删除操作称为 出栈 pop
